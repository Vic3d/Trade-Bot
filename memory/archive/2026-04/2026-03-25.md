## 11:30–13:00 — TradeMind Dashboard JS Syntax Error (HAUPTBUG)
- **Symptom:** Dashboard lädt nicht, alle Panels zeigen "Lädt…"
- **Root Cause:** `\n` in Template-String-Literal im `shortNames`-Objekt:
  ```
  const shortNames = {OPTIONS_FLOW_OIL_SHORT:'Öl Kurzläufer\n(<7 Tage)', ...};
  ```
  Im Server-Template wird `\n` als echter Zeilenumbruch aufgelöst → bricht Single-Quoted String im Client-JS → **SyntaxError: Invalid or unexpected token**
- **Diagnose:** curl → HTML holen → `<script>` extrahieren → `node --check` → Zeile 1331 identifiziert
- **Fix (db52742):** `\n` durch Leerzeichen ersetzt, `.replace('\n','<br>')` entfernt
- **Lektion:** In Server-side Template Literals (`\`...\``) werden `\n` in eingebetteten Strings zu echten Newlines → NIEMALS `\n` in Single-Quoted Strings innerhalb von Template Literals verwenden!
