# Trading Journal

Ein Eintrag pro Trade-Entscheidung. Dateiname: `YYYY-MM-DD-[ticker].md`

## Zweck
- Entscheidungen nachvollziehbar machen (warum gekauft/verkauft?)
- Lernen aus Fehlern (RHM-Lektion: Stop nicht real gesetzt)
- Strategie-Disziplin messen (halte ich meine eigenen Regeln ein?)

## Template

```markdown
# [Datum] — [Aktienname (TICKER)]

## Entscheidung
[ ] Kauf | [ ] Verkauf | [ ] Hold | [ ] Watchlist-Update

## Setup
- Entry: €
- Stop: €
- Ziel: €
- CRV:

## Begründung
(Warum diese Entscheidung? Welches Signal? Welche These?)

## Kontext
(Marktlage, News, Geopolitik zum Zeitpunkt)

## Ergebnis
(Wird nachträglich ausgefüllt)
- Exit: € | Datum:
- P&L: % / €
- Was lief gut / was falsch?
```
