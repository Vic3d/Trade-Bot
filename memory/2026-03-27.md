1. **v2.1.0 noch im Topbar-Span** (nur `<title>` war auf v2.1.1 geändert, nicht `<span>v2.1.0</span>`)

5. **`dna.profile`** — `total_trades` und `win_rate` kommen aus `dna.stats`, nicht aus `dna.trader_profile`
6. **Vercel cached alten kaputten Build** — `node --check` lokal PASS, aber Vercel servierte noch alten fehlerhaften Commit

### Fix-Methode
- Alle Patches via Python `bytes`-Replacement-Scripts (vermeidet `\"` Escaping-Hölle in verschachtelten JS-Strings)
- `node --check /tmp/tmpfile.js` statt stdin (ENOENT-Bug bei stdin-Check in dieser Umgebung)

### Bug #1: Emoji-Escape (\U → kaputte Darstellung)
- **Problem:** Python `b'\U0001f3e0'` in Byte-Strings = Literal-Backslash, nicht Unicode-Escape
  - JavaScript renderte `U0001f3e0` statt 🏠
  - Screenshot von Victor: alle Nav-Tab-Emojis kaputt
- **Fix:** \U-Sequenzen → JS Surrogatpaare (\ud83c\udfe0 etc.) — 19 Stellen gefixt
- **Commits:** `53fb9ed`, dann `d0e9e99` (force Vercel redeploy nötig)
- **Vercel-Problem:** Stale Build → lösung: package.json `_deploy` Timestamp ändern

### Bug #2: Vercel Stale Build Problem (wiederholend)
- **Symptom:** GitHub hat korrekten Code, aber Vercel serviert alte Version (Etag konstant)
- **Ursache:** Vercel cached Build aggressiv; push von nicht-deployten Files ([skip ci] commits) stört Queue
- **Lösung:** `vercel.json` ändern (Header hinzufügen) → garantiert frischer Build
- **Zuverlässigste Methode:** `vercel.json` + `api/dashboard.js` gleichzeitig pushen

### Bug #3: nav-badge IDs fehlen → _render() Crash (HAUPTBUG leere Sections)
- **Problem:** Neuer 7-Tab-Nav hat keine Badge-Spans mehr (`nav-pos`, `nav-strat`, etc.)
  - Aber `_render()` versuchte `$('nav-pos').textContent = ...` → `null.textContent` → TypeError
  - Crash passiert NACH `$('loader').style.display='none'` → Loader unsichtbar, Error unsichtbar
  - Alle 37 innerHTML-Assignments danach nie ausgeführt → alle Sektionen leer
- **Diagnose:** `nav-pos` in Live-HTML: MISSING (Elementcheck mit Python) 
- **Fix:** Null-safe Helper `var nb=function(id,v){var e=$('nav-'+id);if(e)e.textContent=v;};`
- **Commits:** `4c1ede9` (fix), `982c298` (force redeploy), `23bc4d6` (vercel.json + fix final)
- **Verifikation:** `Badge fix live: True` nach deploy
